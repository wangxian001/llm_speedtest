OpenAI兼容接口测试说明

  • 使用流式响应(stream=true)测量首字延迟
  • 预填充速度 = 提示词长度 / (首字到达时间 - 请求开始时间)
  • 输出速度 = 输出token数 / (输出完成时间 - 首字到达时间)
  • 适用于所有兼容OpenAI API的大模型服务
  • 支持思考模型（DeepSeek-R1、Qwen-QwQ等）：自动统计总吞吐（思考+答案tokens）
